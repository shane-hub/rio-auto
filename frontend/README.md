# Rio Auto Frontend

This project is built with Vue 3, TypeScript, Vite, and Element Plus.

## Setup

1. Install dependencies:
   ```bash
   npm install
   ```

2. Run development server:
   ```bash
   npm run dev
   ```

3. Build for production:
   ```bash
   npm run build
   ```

## Configuration

- `vite.config.ts`: Configured to proxy `/api` requests to `http://127.0.0.1:8000` (Backend).
